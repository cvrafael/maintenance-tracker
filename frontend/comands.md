# Abrir o editor:
## Abra o terminal e digite - vim

# Sair do editor
## No modo normal digite - :q ou :q! (caso queira forçar a saída)

# Acessar o modo normal:
## Pressione a tecla Esc

# Modo insert:
## Pressione a tecla i

# Modo visual:
## Pressione a tecla v

# Abrir um arquivo
## vim [nome-do-arquivo]

# Abrir algum arquivo com o Vim em execução:
## :edit [nome-do-arquivo]
## :edit caminho/ate/o/arquivo/[nome-do-arquivo]
## :Explore

# No modo normal é possível:
## Desfazer alteração:
## Pressione u

# Refazer alteração:
## Pressione ctrl+r

# Movimentação simples:
## h = esquerda
## j = baixo
## k = cima
## l = direita

# Colar:
## Pressione p

# No modo visual é possível:
## Copiar:
## Pressione y

# Recortar:
## Pressione d = para algum trecho
## Pressione dd = para a linha inteira

# Salvar arquivo:
## :w
## :w [nome-do-arquivo]
## :w caminho/ate/o/arquivo/[nome-do-arquivo]

# Deletar arquivo:
## :Explore
## Navege até o arquivo
## Pressione D e confirme
